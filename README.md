## :gear: Technologies

This project was developed to make a test of chat in Real Time with Firebase and React Native. The Technologies used:

  - React Native
  - Firebase
  - Expo

## :computer: How to run

  ```bash
    # clone repository
    git clone

    # enter web folder
    cd project-name

    # download dependencies
    yarn or npm install

    # running web project
    expo start
  ```
